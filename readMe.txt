Para utilizar a aplicação é nececssário criar um servidor local.
Utilizando python:
Com  o terminal/powershell na pasta da aplicação:
		python -m http.server
		
Abrir aplicação no browser:
		http://localhost:8000/visApp.html
	

Para actualizar/guardar o dataset deve ser executado o "readSerial.py"	
	
Nota: A aplicação só foi testada em Mozilla, pelo  que sera preferível utiliza-la nesse browser.
